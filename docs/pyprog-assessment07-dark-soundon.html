Formative Assessment - Snakify Lesson 7

This formative assessment tests your knowledge of Snakify Lesson 7 (Lists), which covers the basics of lists in Python programming. The assessment has two parts: an easy section and a slightly harder section, each with 10 multiple-choice questions.

The easy section reviews the fundamental concepts of lists that you learned in Lesson 7. The slightly harder section challenges you with more advanced questions that require deeper understanding of the lesson content.

For each question, you will see a code snippet and four options. Select the option that best answers the question based on your knowledge of the material. You can take your time and read each question carefully before submitting your answer.

We wish you all the best and hope you enjoy learning!

Easy

  1. What is the output of the following code snippet?
    numbers = [1, 2, 3, 4, 5]
print(numbers[1:3])

    a) [3, 4] b) [1, 2] c) [2, 3] d) [2, 4]

    Answer: c) [2, 3]

  1. What is the output of the following code snippet?
    numbers = [1, 2, 3, 4, 5]
print(numbers[-3:-1])

    a) [2, 3] b) [3, 4] c) [4, 5] d) [2, 4]

    Answer: b) [3, 4]

  1. What is the output of the following code snippet?

Slightly Harder

  1. What is the output of the following code snippet?
    nums = [2, 4, 6, 8, 10]
print(nums[1:3])

    a) [2, 4] b) [6, 8] c) [4, 6, 8] b) [4, 6]

    Answer: d) [4, 6]

  1. What is the output of the following code snippet?
    nums = [1, 2, 3, 4, 5]

    a) [1, 3, 4] b) [1, 3, 5] c) [2, 4, 5] d) [2, 3, 4]

    Answer: a) [1, 3, 4]

  1. What is the output of the following code snippet?

nums.append([4, 5])

    Answer: b) 4

    Answer: c) [2, 3, 4]

    Answer: c) [10, 20, 25, 30]

  1. What is the output of the following code snippet?
