【課堂練習 1】Merge two numbers

  • Given two two-digit numbers, merge their digits as shown in the tests below.
  • The input must be processed as integers and must not be converted to strings.
  • It is not allowed to use string concatenation to solve this problem.

a = int(input())
b = int(input())
tens_a = a // 10
units_a = a % 10
tens_b = b // 10
units_b = b % 10
print(tens_a * 1000 + tens_b * 100 + units_a * 10 + units_b)

【課堂練習 4】Distance to closest point

Write a program to calculate the shortest distance between a given point A and either point B or C on a line. Provide the coordinates of all three points as input.

a = int(input())
b = int(input())
c = int(input())

a_to_b = abs(a - b)
a_to_c = abs(a - c)

if a_to_b < a_to_c:
    print(a_to_b)
else:
    print(a_to_c)

【課堂練習 5】Four-digit palindrome

  • Write a program that takes a four-digit integer and checks if it's a palindrome. If it is, print "YES", otherwise print "NO".
  • A palindrome is a number that reads the same forwards as it does backwards.
  • The program must only use numerical data types and must not involve any other data types.
    Test case 1:
Input: 1234
Output: NO

Test case 2:
Input: 3443
Output: YES

Test case 3:
Input: 2222
Output: YES

Test case 4:
Input: 1221
Output: YES

number = int(input())
a = number // 1000
b = number // 100 % 10
c = number // 10 % 10
d = number % 10
if a == d and b == c:
    print('YES')
else:
    print('NO')

a = int(input())

if (a%10!=a//1000):
    print("NO")
elif (a%100//10!=a//100%10):
    print("NO")
else:
    print("YES")
