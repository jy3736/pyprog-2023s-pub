Course Description

This course is an introductory course to Python programming, covering topics such as basic syntax, data types, file processing, exception handling and object-oriented programming. Students will gain a comprehensive understanding of the basics of Python programming by working through specifically designed class exercises and assignments. By the end of the course, students will have developed the ability to use Python to solve practical problems.

  1. Python Basic Syntax:
    • Variable declaration: Understanding and using various data types such as strings, numbers(integer, float), booleans, etc. in Python. Assigning values to variables and naming conventions.
    • Operators: Arithmetic, comparison, and logical operators, etc. Understanding the usage of each operator and how they work in different scenarios.
    • Control Flow: Conditional statements(if-else), loops(for, while) and their usage in controlling the flow of a program. Understanding how to create flow control in a program using different statements.
    • Function: How to define and call functions in Python. Understanding the syntax for creating a function, passing arguments, and returning values.
    • Data Types: Understanding the different types of data types available in Python, such as basic data types (strings, numbers, booleans), and collection data types (lists, tuples, dictionaries). Understanding the characteristics and operations of each data type.
  1. File Processing:
    • File handling: Understanding how to open, read, and write files in Python. Understanding the different modes of opening a file (e.g. read, write, append) and the various file operations.
    • File System: Understanding how to navigate and manipulate the file system using Python. Understanding how to create, rename, delete, and move files and directories.
  1. Error and Exception Handling:
    • Understanding how to handle errors and exceptions in Python. Understanding how to use try-except blocks to catch and handle exceptions, and how to use the raise statement to throw exceptions.
    • Understanding the difference between exceptions and errors, and how to handle each appropriately.
  1. Object-Oriented Programming (OOP): [Optional]
    • Understanding the concepts of classes, objects, inheritance, polymorphism, etc. Understanding the benefits of OOP and how it is used in Python.
    • Understanding how to create and use classes and objects in Python. Understanding how to define and use class variables, class methods, and instance methods.
