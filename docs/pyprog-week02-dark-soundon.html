【課堂練習 2】下載並安裝 Python Interperter 和 Visual Studio Code

  • Download/Install Python Interpreter:
    1. Go to the official Python website (https://www.python.org/) and click on the "Download" button. Download the latest version for your operating system (Windows/macOS/Linux).
    1. Run the installer and follow the prompts to install Python on your computer. Make sure to select the option to add Python to your system's PATH, so that it can be accessed from the command line.
  • Download/Install Visual Studio Code:
    1. Go to the Visual Studio Code website (https://code.visualstudio.com/) and click on the "Download" button. Select the appropriate version for your operating system and download the installer.
    1. Run the installer and follow the prompts to install Visual Studio Code on your computer. Once installation is complete, launch the program and you are ready to start coding.
    1. Install Python Extension: Once you have Visual Studio Code open, press F1 or Ctrl+Shift+P (Windows/Linux) or Cmd+Shift+P (macOS), then type "Python" and select the option "Python: Select Interpreter" and then select the python version you have installed.

【作業 2】Write a program that takes a three-digit number as input and prints its digits separately.

  • Please refer to the example for "two-digit" processing below and complete the task requirements.
number = int(input("Enter a two-digit number: "))

# using integer division (//) and modulo operator (%)
tens = number // 10
ones = number % 10

print("The tens digit is:", tens)
print("The ones digit is:", ones)
