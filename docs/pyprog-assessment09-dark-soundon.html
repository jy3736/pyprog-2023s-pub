Formative Assessment - Snakify Lesson 9

This assessment measures your understanding of Snakify Lesson 9 (Arrays), which introduces the basics of lists in Python programming. The assessment consists of two parts: a basic section and an intermediate section, each with 10 multiple-choice questions. The basic section covers the essential concepts of lists that you learned in Lesson 9. The intermediate section tests your skills with more complex questions that demand deeper understanding of the lesson content.

For each question, you will see a code snippet and four options. Choose the option that best answers the question based on your knowledge of the material. You can work at your own pace and read each question carefully before submitting your answer. We hope you have fun and learn a lot!

Easy

  1. What is the output of the following code snippet?
    arr = [[1, 2], [3, 4], [5, 6]]
print(arr[1][0])

    A. 1 B. 2 C. 3 D. 4

    Answer: C

  1. What is the output of the following code snippet?
    arr = [[1, 2], [3, 4], [5, 6]]
print(arr[2][-1])

    A. 1 B. 2 C. 5 D. 6

    Answer: D

  1. What is the output of the following code snippet?
    arr = [[1, 2], [3, 4], [5, 6]]
arr[1][1] = 0
print(arr)

    A. [[1, 2], [3, 4], [5, 6]] B. [[1, 2], [3, 0], [5, 6]] C. [[1, 2], [0, 4], [5, 6]] D. [[1, 0], [3, 4], [5, 6]]

    Answer: B

    arr = [[1, 2], [3, 4], [5, 6]]

    Answer: A

  1. What is the output of the following code snippet?

  1. What is the output of the following code snippet?
    arr = [[i + j for j in range(3)] for i in range(4)]

print(new_arr)

    A. [2, 4] B. [1, 3, 5, 7] C. [0, 2, 4, 6] D. [1, 2, 3, 4]

    Answer: A

  1. What is the output of the following code snippet?

print(arr)

    A. [[1, 2], [3, 4]] B. [[4, 8], [12, 16]] C. [[2, 4], [6, 8]] D. [[2, 6], [4, 8]]

    Answer: C

  1. What is the output of the following code snippet?

  1. What is the output of the following code snippet?

  1. What is the output of the following code snippet?
