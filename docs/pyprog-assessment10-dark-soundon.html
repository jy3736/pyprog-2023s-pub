Formative Assessment - Snakify Lesson 10

This assessment evaluates your comprehension of Snakify Lesson 10, which focuses on sets and introduces fundamental concepts of lists in Python programming. The assessment comprises two sections: a basic section and an intermediate section, each containing ten multiple-choice questions. The basic section covers the core concepts of sets learned in Lesson 10, while the intermediate section challenges your understanding of the lesson material with more intricate questions.

Each question consists of a code snippet and four possible answers. Based on your knowledge of the material, choose the option that best answers the question. You can complete the assessment at your own pace and carefully review each question before submitting your answer. We hope you enjoy the assessment and acquire new knowledge!

Easy

  1. What is the output of the following code snippet?
    set1 = set([1, 2, 3])
set2 = set([2, 3, 4])
result = set1.union(set2)
print(len(result))

    A) 2 B) 3 C) 4 D) 5

    Correct answer: C) 4

  1. What is the output of the following code snippet?
    set1 = set([1, 2, 3])
set2 = set([2, 3, 4])
result = set1.intersection(set2)
print(result)

    A) {1, 4} B) {2, 3} C) {2, 3, 4} D) {1, 2, 3, 4}

    Correct answer: B) {2, 3}

  1. What is the output of the following code snippet?

    Correct answer: D) {1}
