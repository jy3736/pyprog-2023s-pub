【課堂練習 5】Number of primes in range

  • Given two integers A and B, determine the count of prime numbers between A and B, including A and B if they are prime.
  • A prime number is defined as a natural number greater than 1 that is only divisible by 1 and itself, with no other positive factors.
  • Print the total number of prime numbers found in the given range.
import math

def count_primes(A, B):
    count = 0
    for num in range(A, B+1):
        if num > 1:
            for i in range(2, int(math.sqrt(num))+1):
                if num % i == 0:
                    break
            else:
                count += 1
    return count

a = int(input())
b = int(input())

print(count_primes(a,b))
