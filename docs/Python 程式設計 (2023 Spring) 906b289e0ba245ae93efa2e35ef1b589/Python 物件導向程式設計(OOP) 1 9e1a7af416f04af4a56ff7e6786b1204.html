„ÄêË™≤Â†ÇÁ∑¥Áøí 1„ÄëDefining and Using Classes

Class Declaration:

# Define the class
class Student:
    def __init__(self, name, age, major):
        self.name = name
        self.age = age
        self.major = major

    # Define a method to print student information
    def print_info(self):
        print(f"Name: {self.name}")
        print(f"Age: {self.age}")
        print(f"Major: {self.major}")

In this section, we define the Student class with its attributes (name, age, and major) and its method (print_info). The __init__ method is a special method in Python that is called when an object is created from the class.

How to use the class to create objects:

# Create instances of the class
student1 = Student("Alice", 21, "Computer Science")
student2 = Student("Bob", 19, "Mathematics")

# Access the properties of the instances
print(student1.name)   # Output: Alice
print(student2.major)  # Output: Mathematics

# Call the methods of the instances
student1.print_info()  # Output: Name: Alice, Age: 21, Major: Computer Science
student2.print_info()  # Output: Name: Bob, Age: 19, Major: Mathematics

In this section, we create two instances of the Student class (student1 and student2) and access their properties (name and major) using the dot notation. We also call the print_info method on both instances to print their information to the console.

class Student:
    def __init__(self, name, age, major, email):
        self.name = name
        self.age = age
        self.major = major
        self.email = email

    # Getter methods to access each property
    def get_name(self):
        return self.name

    def get_age(self):
        return self.age

    def get_major(self):
        return self.major

    def get_email(self):
        return self.email

    # Setter methods to modify each property
    def set_name(self, name):
        self.name = name

    def set_age(self, age):
        self.age = age

    def set_major(self, major):
        self.major = major

    def set_email(self, email):
        self.email = email

Utilization:

student1 = Student("John Doe", 20, "Computer Science", "john.doe@example.com")

class Complex:
    def __init__(self, real, imag):
        self.real = real
        self.imag = imag
    
    def add(self, other):
        real_sum = self.real + other.real
        imag_sum = self.imag + other.imag
        return Complex(real_sum, imag_sum)

    def subtract(self, other):
        real_diff = self.real - other.real
        imag_diff = self.imag - other.imag
        return Complex(real_diff, imag_diff)

    def multiply(self, other):
        real_prod = self.real * other.real - self.imag * other.imag
        imag_prod = self.real * other.imag + self.imag * other.real
        return Complex(real_prod, imag_prod)

    def divide(self, other):
        divisor = (other.real ** 2 + other.imag ** 2)
        real_num = self.real * other.real + self.imag * other.imag
        imag_num = self.imag * other.real - self.real * other.imag
        return Complex(real_num/divisor, imag_num/divisor)

    def __str__(self):
        return "{} + {}i".format(self.real, self.imag)

Class Usage:

# Create two Complex numbers
c1 = Complex(2, 3)
c2 = Complex(4, 5)

# Add two Complex numbers
result = c1.add(c2)
print(result.real, result.imag) # Output: 6 8

# Subtract two Complex numbers
result = c1.subtract(c2)
print(result.real, result.imag) # Output: -2 -2

# Multiply two Complex numbers
result = c1.multiply(c2)
print(result.real, result.imag) # Output: -7 22

# Divide two Complex numbers
result = c1.divide(c2)
print(result.real, result.imag) # Output: 0.5609756097560976 -0.0487804878048781

„ÄêË™≤Â†ÇÁ∑¥Áøí 4„ÄëComplex Class with Operator Overloading

Class Declaration:

In this section, we define the Complex class which represents a complex number. We declare the class with the class keyword followed by the class name. The class has two attributes real and imag which represent the real and imaginary parts of the complex number. We define the __init__ method which is a constructor that initializes the real and imag attributes when a new object of the class is created. We also define the __str__ method which returns a string representation of the object.

class Complex:
    def __init__(self, real, imag):
        self.real = real
        self.imag = imag

    def __str__(self):
        return "{} + {}i".format(self.real, self.imag)

    def __add__(self, other):
        return Complex(self.real + other.real, self.imag + other.imag)

    def __sub__(self, other):
        return Complex(self.real - other.real, self.imag - other.imag)

    def __mul__(self, other):
        return Complex(self.real * other.real - self.imag * other.imag,
                       self.real * other.imag + self.imag * other.real)

    def __truediv__(self, other):
        denominator = other.real ** 2 + other.imag ** 2
        return Complex((self.real * other.real + self.imag * other.imag) / denominator,
                       (self.imag * other.real - self.real * other.imag) / denominator)

Class Usage:

In this section, we use the Complex class to create objects and perform some arithmetic operations with them. We create two Complex objects a and b by calling the class constructor with the real and imaginary parts as arguments. We then use the __add__, __sub__, __mul__, and __truediv__ methods to perform addition, subtraction, multiplication, and division operations between the a and b objects.

a = Complex(1, 2)
b = Complex(3, 4)

print("a = {}".format(a))
print("b = {}".format(b))
print("a + b = {}".format(a + b))
print("a - b = {}".format(a - b))
print("a * b = {}".format(a * b))
print("a / b = {}".format(a / b))

The output of the above code will be:

a = 1 + 2i
b = 3 + 4i
a + b = 4 + 6i
a - b = -2 - 2i
a * b = -5 + 10i
a / b = 0.44 + 0.08i

Here, we use the print() function to print the string representations of the Complex objects and the results of the arithmetic operations to the console. We use the format() method to insert the values of the objects and the results into the string.
