Snakify Lesson 5 is about strings, which are sequences of characters that can be defined using single or double quotes. The lesson covers how to read strings from the standard input using the input() function, how to access individual characters or substrings using indexing and slicing, how to use string methods such as len(), lower(), upper(), count(), find(), replace() and strip(), how to iterate over strings using for loops, and how to format strings using placeholders and the format() method. 【課堂練習 1】Problem «The two halves»

  • Cut the given string into two parts of equal length (if the length of the string is odd, the middle character should be in the first part, which will contain one more character than the second part).
  • Print a new string with the second half followed by the first half.
  • Don't use if statement to solve this task.
    from math import ceil

s = input()
c = ceil(len(s)/2)
print(s[c:]+s[:c])
s = input()
c = -(-len(s)//2)
print(s[c:]+s[:c])

【課堂練習 2】Problem «The second occurrence»

  • If a string contains the letter 'f', print the index of its second occurrence.
  • If it only appears once, print -1, if it does not appear at all, print -2.
s = input()
if s.count('f') == 1:
    print(-1)
elif s.count('f') < 1:
    print(-2)
else:
    print(s.find('f', s.find('f') + 1))
def second_f(s):
    count = s.count("f")
    if count < 2:
        return -1 if count == 1 else -2
    first_f = s.index("f")
    s = s[first_f+1:]
    return s.index("f") + first_f + 1

print(second_f(input()))

【課堂練習 3】Problem «Reverse the fragment»

  • If a string contains the letter 'h' at least twice, reverse the order of characters between the first and last occurrences of 'h'.
    s = input()
a = s[:s.find('h')] 
b = s[s.find('h'):s.rfind('h') + 1]
c = s[s.rfind('h') + 1:]
s = a + b[::-1] + c
print(s)
def reverse_h(s):
    first_h = s.index("h")
    last_h = s.rindex("h")
    if first_h != last_h:
        s = s[:first_h + 1] + s[first_h + 1:last_h][::-1] + s[last_h:]
    return s

print(reverse_h(input()))

【課堂練習 5】Problem «The index of the maximum of a sequence»

  • Given a sequence of integers that ends with the number 0, determine the index of the largest number in the sequence.
  • If multiple numbers are tied for the largest value, print the index of the first occurrence of that value.
  • The returned index value should begin at 1.
    max = 0
index_of_max = -1
element = -1
len = 1
while element != 0:
    element = int(input())
    if element > max:
        max = element
        index_of_max = len
    len += 1
print(index_of_max)
def max_idx(lst):
    m = max(lst)
    if lst.count(m) == 1:
        return lst.index(m)+1
    for i in range(len(lst)):
        if lst[i] == m:
            return i+1

l =[]
while True:
    c = int(input())
    if c==0: break
    l.append(c)

print(max_idx(l))

【課堂練習 6】Problem «The second maximum»

  • Given a sequence of distinct, positive integers that ends with the number 0, determine the value of the second largest number in the sequence.
  • It is guaranteed that the sequence contains at least two elements.
    first_max = int(input())
second_max = int(input())
if first_max < second_max:
    first_max, second_max = second_max, first_max
element = int(input())
while element != 0:
    if element > first_max:
        second_max, first_max = first_max, element
    elif element > second_max:
        second_max = element
    element = int(input())
print(second_max)
def sec_largest(arr):
    m1 = max(arr)
    arr.remove(m1)
    m2 = max(arr)
    return m2
    
def read_seq():
    seq = []
    num = int(input())
    while num != 0:
        seq.append(num)
        num = int(input())
    return seq

print(sec_largest(read_seq()))

【課堂練習 7】Problem «The maximum number of consecutive equal elements»

  • Given a sequence of integer numbers that ends with the number 0, determine the longest consecutive stretch of numbers where all elements are the same.
prev = -1
curr_rep_len = 0
max_rep_len = 0
element = int(input())
while element != 0:
    if prev == element:
        curr_rep_len += 1
    else:
        prev = element
        max_rep_len = max(max_rep_len, curr_rep_len)
        curr_rep_len = 1
    element = int(input())
max_rep_len = max(max_rep_len, curr_rep_len)
print(max_rep_len)
def max_clen(sequence):
    max_length = 0
    curr_length = 0
    curr_num = None
    for num in sequence:
        if num == curr_num:
            curr_length += 1
        else:
            curr_num = num
            curr_length = 1
        max_length = max(max_length, curr_length)
    return max_length
    
def read_seq():
    seq = []
    num = int(input())
    while num != 0:
        seq.append(num)
        num = int(input())
    return seq

print(max_clen(read_seq()))
