Formative Assessment - Snakify Lesson 8

This formative assessment is designed to test your understanding of Snakify Lesson 8 (functions). The assessment consists of two parts: easy and harder. The easy part covers the basic concept of functions, such as how to define, call, and return values from them. The harder part requires more comprehensive knowledge application, such as how to use parameters, arguments, scope, and recursion. Each part has 10 multiple-choice questions based on the knowledge learned from the class and Snakify online material.

You should complete this assessment after finishing Snakify Lesson 8 (functions) and reviewing the relevant class notes. This assessment will help you prepare for the summative assessment and improve your programming skills.

Easy

print(result)

    Answer: C

    return count

    Answer: B

  1. What is the output of the following code snippet?
    def is_palindrome(string):
    if string == string[::-1]:
        return True
    else:
        return False

words = ["radar", "level", "hello", "racecar"]
palindromes = filter(is_palindrome, words)
result = list(palindromes)
print(result)

    A. ["radar", "level", "racecar"] B. ["hello"] C. ["level"] D. ["radar", "level", "hello", "racecar"]

    Answer: A
