Formative Assessment - Snakify Lesson 11

This assessment tests your understanding of Snakify Lesson 11 on dictionaries in Python. It has two sections: basic and intermediate, with ten multiple-choice questions each. The basic section reviews the main concepts of dictionaries from Lesson 11, while the intermediate section applies them to more complex scenarios.

Each question shows a code snippet and four options. Choose the best option based on your knowledge. You can work at your own pace and check each question carefully before answering. We hope you learn something new and have fun!

Easy

  1. What is the output of the following code snippet?
    d = {"apple": 3, "banana": 1, "cherry": 2}
del d["banana"]
print(len(d))

    A. 2 B. 1 C. 3 D. 0

    Answer: A

  1. What is the output of the following code snippet?
    d = {"a": 1, "b": 2, "c": 3}
d["b"] += 1
print(d["b"])

    A. 1 B. 3 C. 2 D. 4

    Answer: B

  1. What is the output of the following code snippet?
    d = {"cat": 4, "dog": 2, "bird": 1}
d["cat"] = 3
print(d)

    A. {'cat': 4, 'dog': 2, 'bird': 1} B. {'cat': 3, 'dog': 2, 'bird': 1} C. {'cat': 3} D. {'cat': 4}

    Answer: B

  1. What is the output of the following code snippet?
    d1 = {"a": 1, "b": 2}
d2 = {"c": 3, "d": 4}
d1.update(d2)
print(len(d1))

    A. 2 B. 3 C. 4 D. 5

    Answer: C

  1. What is the output of the following code snippet?
    d = {"a": 1, "b": 2, "c": 3}
del d["d"]
print(len(d))

    A. 1 B. 2 C. 3 D. Trigger a KeyError exception

    Answer: D

  1. What is the output of the following code snippet?
    d = {"apple": 1, "banana": 2, "cherry": 3}
print(d.get("orange", 0))

    A. 1 B. 0 C. None D. Trigger a KeyError exception

    Answer: B

  1. What is the output of the following code snippet?
    d = {"apple": 1, "banana": 2, "cherry": 3}
d.popitem()
print(len(d))

    A. 0 B. 1 C. 2 D. 3

    Answer: C

  1. What is the output of the following code snippet?
    d = {"apple": 1, "banana": 2, "cherry": 3}
print("apple" in d)

    A. True B. False C. None D. Trigger a KeyError exception

    Answer: B

  1. What is the output of the following code snippet?
    d = {"a": 1, "b": 2, "c": 3}
for key in d.keys():
    print(key, end=" ")

    A. a b c B. 1 2 3 C. {'a', 'b', 'c'} D. ['a', 'b', 'c']

    Answer: A

  1. What is the output of the following code snippet?
    d = {"a": 1, "b": 2, "c": 3}
print(sum(d.values()))

    A. 6 B. 2 C. 1 D. 3

    Answer: A

Slightly Harder

  1. What is the output of the following code snippet?
    dict_1 = {'a': 1, 'b': 2, 'c': 3}
dict_2 = {'b': 3, 'c': 4, 'd': 5}
dict_1.update(dict_2)
print(dict_1['c'])

    A. 3 B. 4 C. 5 D. This code will produce an error.

    Answer: B

  1. What is the output of the following code snippet?
    my_dict = {'a': [1, 2], 'b': [3, 4], 'c': [5, 6]}

for key, value in my_dict.items():
    value.append(7)

print(my_dict['b'])

    A. [3, 4] B. [3, 4, 7] C. [5, 6] D. [5, 6, 7]

    Answer: B

  1. What is the output of the following code snippet?
    my_dict = {'a': 1, 'b': 2, 'c': 3}
del my_dict['b']
print('b' in my_dict.keys())

    A. True B. False C. This code will produce an error. D. This code will produce an empty output.

    Answer: B

  1. What is the output of the following code snippet?
    my_dict = {'a': 1, 'b': 2, 'c': 3}
my_list = list(my_dict)
print(my_list)

    A. ['a', 'b', 'c'] B. [1, 2, 3] C. [('a', 1), ('b', 2), ('c', 3)] D. This code will produce an error.

    Answer: A

  1. What is the output of the following code snippet?
    my_dict = {'a': 1, 'b': 2, 'c': 3}

def my_function(d):
    d['a'] = 100

my_function(my_dict)
print(my_dict['a'])

    A. 1 B. 2 C. 3 D. 100

    Answer: D

my_set = set(my_dict.keys())
print(len(my_set))

    A. 1 B. 2 C. 3 D. This code will produce an error.

    Answer: C

my_dict.update({'a': 100})

my_dict.clear()
print(len(my_dict))

def my_function(d):

my_function(my_dict)
print(len(my_dict))

  1. What is the output of the following code snippet?
    my_dict = {'a': 1, 'b': 2, 'c': 3}
my_dict.pop('b')
print(my_dict)

    A. {'a': 1, 'c': 3} B. {'a': 1, 'b': 2} C. {'b': 2, 'c': 3} D. {'b': 2}
